## About

A clean, responsive personal portfolio site built to present my work to recruiters and collaborators. It covers who I am, what I've built, and how to reach me.

---

## Sections

| Section | Description |
|---|---|
| **Hero** | Introduction, tagline, CTA buttons (View Projects / Contact) |
| **About** | Background, education, and what I'm currently focused on |
| **Skills** | Tech stack with visual grouping (Languages, Frameworks, Tools) |
| **Projects** | Cards for each major project with links to GitHub repos |
| **Resume** | Downloadable resume (PDF) |
| **Contact** | Email and social links |

## Setup & Run Locally

```bash
git clone https://github.com/Rajnish7569/portfolio.git
cd portfolio
# Just open index.html in your browser — no build step needed
open index.html
```

---

## Deployment (GitHub Pages)

```bash
# Push to main branch — GitHub Pages auto-deploys from root
git add .
git commit -m "feat: update portfolio"
git push origin main
```

Then go to repo **Settings → Pages → Source → main branch → Save**.
Your site will be live at `https://rajnish7569.github.io/portfolio`
